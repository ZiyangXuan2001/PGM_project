你的担心是对的。

**如果方案 1 只写成**
[
\tilde Y_t=f_\theta(X_t),\qquad \sum_t|\tilde Y_t-\tilde Y_{t-1}|^2,\qquad \hat C=g_\omega(\tilde Y_{1:T})
]
那它的确更像：

* neural encoder
* temporal smoothness regularization
* classifier

这时候 **PGM 味道不够强**。

但关键是：

# PGM 不在 (X\to Y) 这个 encoder 里

# PGM 在“给定 (X) 后，如何对整条 (Y_{1:T}) 做结构化推断”里

也就是说，**神经网络可以只是提供局部证据**，真正的 PGM 部分是后面的 hidden trajectory inference。

## 你现在这个 energy，其实已经可以变成真正的图模型

你写的
[
E(Y)=\sum_t |Y_t-\tilde Y_t|^2+\lambda\sum_{t=2}^T|Y_t-Y_{t-1}|^2
]
不是随便写的 loss。

它其实对应一个条件图模型：

[
p(Y_{1:T}\mid X_{1:T})
\propto \exp\big(-E(Y)\big)
]

把它展开看：

### unary factor

[
\phi_t(Y_t,X_t)=\exp\left(-\frac{1}{2\sigma_x^2}|Y_t-\tilde Y_t|^2\right)
]
其中
[
\tilde Y_t=f_\theta(X_t)
]

### pairwise factor

[
\psi_t(Y_{t-1},Y_t)=\exp\left(-\frac{1}{2\sigma_t^2}|Y_t-Y_{t-1}|^2\right)
]

于是：
[
p(Y_{1:T}\mid X_{1:T})
\propto
\prod_{t=1}^T \phi_t(Y_t,X_t)\prod_{t=2}^T \psi_t(Y_{t-1},Y_t)
]

这已经是一个**链式 pairwise MRF / continuous CRF / Gaussian-style graphical model**了。

所以：

* (f_\theta(X_t)) 不是整个模型
* 它只是 unary potential 的参数化方式
* 真正的 PGM 是整条 (Y) 的联合结构和 MAP inference

---

## 所以你可以把 (X\to Y) 改名，不要叫它“直接输出 hidden state”

这一步很重要。

不要说：

[
Y_t=f_\theta(X_t)
]

更好的说法是：

[
Z_t=f_\theta(X_t)
]

这里 (Z_t) 是：

* proposal
* pseudo-observation
* local evidence
* unary evidence point

然后真正的 hidden variable 是 (Y_t)。

也就是：

### Step 1

[
Z_t=f_\theta(X_t)
]

### Step 2

[
p(Y_{1:T}\mid Z_{1:T})
\propto
\prod_t \phi_t(Y_t,Z_t)\prod_t \psi_t(Y_{t-1},Y_t)
]

### Step 3

[
Y^*=\arg\max_Y p(Y\mid Z)
]

### Step 4

[
\hat C=g_\omega(Y^*)
]

这样一下就清楚了：

* NN 只负责把 frame tokens 变成局部证据 (Z_t)
* PGM 负责把这些局部证据整合成全局一致的 hidden trajectory (Y_{1:T})

这个区别非常关键。

---

## 这时你的项目和 PGM 的关系就很正了

因为你在做的是：

### 1. hidden variables

[
Y_1,\dots,Y_T
]

### 2. graph structure

[
Y_{t-1}\leftrightarrow Y_t
]

### 3. unary + pairwise factors

[
\phi_t(Y_t,Z_t),\qquad \psi_t(Y_{t-1},Y_t)
]

### 4. inference

[
Y^*=\arg\min_Y E(Y)
]

这已经不是普通 smoothness loss 了，而是**图上的 MAP inference**。

## 你现在的问题本质上是：

> “如果前面用了神经网络，是不是就不算 PGM 了？”

答案是：

**不是。**

现代很多 structured model 都是：

* 图结构是 PGM
* factor/potential 用神经网络参数化

也就是：

* 传统 PGM：手写 factor table
* 现代 PGM：用 NN 输出 unary score / observation proposal

核心没变，变的是 factor 的参数化方式。

## 但你这个版本，PGM 强度确实有层次差别

### 弱 PGM

[
\tilde Y_t=f_\theta(X_t)
]
然后直接加 smoothness loss

这更像 regularization。

### 中等 PGM

引入 hidden (Y_t)，并定义
[
p(Y\mid X)\propto \exp(-E(Y))
]
然后做
[
Y^*=\arg\min E(Y)
]

这个已经是实打实的 structured inference。

### 强 PGM

再把它写成明确的 probabilistic observation + transition model：

[
Z_t=f_\theta(X_t)
]
[
p(Z_t\mid Y_t)=\mathcal N(Y_t,\sigma_x^2 I)
]
[
p(Y_t\mid Y_{t-1})=\mathcal N(AY_{t-1},Q)
]

然后做 smoothing / MAP / posterior covariance。

这个就更像经典连续状态图模型了。
